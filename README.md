# Gaussian Plot Enhancements 📊
A Python-based visualization project that generates and enhances Gaussian (normal distribution) plots using NumPy and Matplotlib, with customizable parameters like mean, standard deviation, and curve styling

---

## 🚀 Features

- Generate Gaussian (Normal) distribution curve
- Adjustable mean (μ) and standard deviation (σ)
- Clean and enhanced Matplotlib visualization
- Easy-to-understand Python implementation
- Grid, labels, and legend for better readability

---
